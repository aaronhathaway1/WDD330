Week 9 Notes

Chapter 9 The Window Object Notes

  1. The Browser Object Model(BOM) is a collection of properties and methods that contain info about the browser and screen
  2. These properties and methods are available through the window object
  3. Global variables are created with const, let or var
  4. These variables can be accessed in all parts of the program
  5. Global variables are properties of the global object
  6. Other examples of methods of the global object are parseInt() and isNaN()
  7. However, ES6 made those methods part of the Number object
  8. alert(), confirm() and propmt() are also methods of the window object
  9. These pop up windows should not be relied upon as they can be disabled by the user
  10. The window object has a navigator property that returns a reference to the navigator object
  11. The window.location property is an object that contains information about the url of the current page
  12. The window.history property shows information about previously visited pages
  13. The window.history.forward() and window.history.back() are used to move the page forward and backward, respectively
  14. A new window can be opened using window.open()
  15. The close() method can close a window
  16. The window.screen object has info on the screen the browser is displayed on
  17. Each window object also contains a document object
  18. Cookies are small files saved locally on a user computer
  19. Cookies take the form of a text file that contains name/value pairs separated by semicolons
  20. This is an example of a cookies file: "name=Superman; hero=true; city=Metropolis"
  21. To create a cookie, you assign it to JS's cookie jar, using the document.cookie property
  22. document.cookie = 'name=Superman';
  23. Assigning more values to document.cookie appends them to the original info
  24. A cookie's value can be reassigned by using the same name but a different value
  25. We can access the contents of the cookie jar by using the split() method to use a for of loop
  26. To remove a cookie, set it to expire:"document.cookie = 'name=Batman; expires=Thu, 01 Jan 1970"
  27. Consider using CSS to animate
