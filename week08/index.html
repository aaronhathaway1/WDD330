Week 8 Notes

Chapter 8 Transforms and Transitions Notes

  1. With CSS3 we can change the appearance of elements
  2. The transform property allows us to translate, rotate, scale and skew any elements on the page
  3. We will not cover 3D forms in this book
  4. Translation allows us to move things along the x and y axis
  5. transform: translate(45px, -45px);
  6. You can specify translateX or translateY to move only 1 direction
  7. We can scale text with scale property
  8. transform: scale(1.5, 0.25);
  9. The rotate function rotates and element around the point of origin by a specified angle value
  10. transform: rotate(10deg)
  11. The skew function specifies a skew along the x and y axes
  12. transform: skew(15deg, 4deg)
  13. Just like translate, you can use skewX and skewY to skew along only one axis
  14. We can transform the origin of an element with "transform-origin: 0 0;
  15. The order of instructions for transforming elements does matter
  16. CSS3 transformations are unsupported for IE.9 and before
  17. You can use postiton: relative to mimics a transform, however
  18. Transitions can also be used to enhance UI and user experience
  19. We can use 'transition-property: transform' to specify what we want to transition
  20. We can use ' transition-duration: 0.2s' to specify how fast we want the transition to take place
  21. 200ms seems to be an optimal transition time, Goldilocks zone

Chapter 12 Canvas, SVG and Drag and Drop Notes

  1. Canvas allows us to manipulate onscreen graphics through JS
  2. First step is to create a canvas element
  3. Grab that element through JS by using an id or class
  4. Also add a width and height attribute to the canvas in the HTML, not the CSS
